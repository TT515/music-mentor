REAPER

Full integration — chat inside the DAW

A native chat panel inside REAPER. One button renders every track, uploads stems + master mix + fader/pan/mute metadata, then you chat with the mentor without leaving the DAW. Add reference tracks from disk to compare against.

⬇ MusicMentor_Chat.lua — chat panel ⬇ MusicMentor_SendSession.lua — send to web app
Install instructions
  1. Chat panel only: install ReaPack (reapack.com — put the .dylib/.dll for your platform in REAPER's resource path → UserPlugins, restart), then Extensions → ReaPack → Browse packages → search imgui → install ReaImGui → Apply → restart.
  2. REAPER → Actions → Show action list → New action → Load ReaScript… → pick the downloaded .lua file.
  3. Run it (assign a keyboard shortcut for one-keystroke access). macOS/Linux; Windows users: use SendSession + the web app.

Logic Pro

Semi-automatic — one click in Logic's export sheet

Logic has no plugin/scripting API for session access, so this helper app automates everything around Logic's own export sheet: it opens the export menu for you, waits for the render, uploads all stems, and opens the mentor. Tick “Include Volume/Pan Automation” at export so the stems carry your mix balance.

⬇ MusicMentor_Logic.applescript ⬇ mentor-send.sh — uploader (required)
Install instructions
  1. Save both files. In Terminal: chmod +x mentor-send.sh, and if needed edit the uploaderPath at the top of the AppleScript to point at it.
  2. Compile the app: osacompile -o "$HOME/Applications/MusicMentor for Logic.app" MusicMentor_Logic.applescript
  3. First run: allow it under System Settings → Privacy & Security → Accessibility, and permit control of Logic Pro.
  4. Flow: open project → run app → pick MusicMentorDrop folder + Export in Logic's sheet (remembered next time) → stems upload → mentor opens.

FL Studio

Export + one command

FL exposes no session API, so the flow is: export split mixer tracks, then run the uploader. FL bakes mixer levels into stems by default, so the mentor hears your actual balance.

⬇ mentor-send.sh
Install instructions
  1. FL Studio: File → Export → Wave file… → tick Split mixer tracks → export to an empty folder.
  2. Terminal: chmod +x mentor-send.sh (first time), then ./mentor-send.sh /path/to/stems "My session".
  3. The mentor opens in your browser with every stem loaded.

All plugins are open source — github.com/TT515/music-mentor.

Why no “real” VST/AU plugin? Plugin formats are sandboxed to a single track's audio and cannot see your session — session-level integration requires each DAW's own scripting surface, which is exactly what these use.
